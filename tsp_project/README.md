# Travelling Salesman Problem (TSP) Solver

هذا المشروع عبارة عن تطبيق متكامل بلغة Python لحل مشكلة البائع المتجول (TSP) باستخدام مجموعة متنوعة من الخوارزميات. يهدف المشروع إلى توفير بيئة للمقارنة بين الخوارزميات الدقيقة والتقريبية من حيث جودة الحل وسرعة التنفيذ، مع توفير تصويرات بيانية توضح مسارات الحلول وأداء كل خوارزمية.

## الخوارزميات المدعومة

يتضمن المشروع تنفيذًا لأربع خوارزميات مختلفة:

1. **Brute Force (القوة الغاشمة):** خوارزمية دقيقة تقوم بتجربة جميع المسارات الممكنة للعثور على الحل الأمثل. نظرًا لتعقيدها الزمني $O(n!)$، فهي مقتصرة على عدد قليل من المدن (10 مدن أو أقل).
2. **Nearest Neighbor (الجار الأقرب):** خوارزمية جشعة (Greedy) سريعة جدًا $O(n^2)$، لكنها لا تضمن الوصول إلى الحل الأمثل.
3. **Simulated Annealing (التلدين المحاكى):** خوارزمية بحث ميتا-هيوريستك مستوحاة من عملية التلدين في علم المعادن. توفر توازنًا ممتازًا بين سرعة التنفيذ وجودة الحل.
4. **Genetic Algorithm (الخوارزمية الجينية):** خوارزمية تطورية تعتمد على مبادئ الانتقاء الطبيعي والوراثة. قادرة على إيجاد حلول قريبة من المثالية للمشاكل المعقدة.

## طريقة التشغيل

لتشغيل المشروع وتوليد النتائج والتصويرات البيانية، قم بتنفيذ الملف الرئيسي:

```bash
python main.py
```

سيقوم البرنامج بتوليد مدن عشوائية، وحل المشكلة باستخدام جميع الخوارزميات، ثم طباعة جدول مقارنة في سطر الأوامر، وحفظ الصور البيانية في مجلد `results`.

## التصويرات البيانية

يقوم المشروع بتوليد عدة صور بيانية في مجلد `results`:
- `*_solution.png`: صورة توضح المسار الذي وجدته كل خوارزمية.
- `comparison_all_algorithms.png`: صورة مجمعة تقارن بين مسارات جميع الخوارزميات.
- `performance_chart.png`: رسم بياني شريطي يقارن بين المسافات وأوقات التنفيذ.
- `convergence_curves.png`: منحنى يوضح سرعة تقارب الخوارزميات التكرارية (SA و GA) نحو الحل الأمثل.
